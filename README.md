# AI Music Recommender

An intelligent music recommendation app powered by OpenRouter AI and Last.fm API. Get personalized song recommendations based on your interests, mood, and favorite artists.

## Features

- AI-Powered Recommendations - Uses OpenRouter's language model to generate intelligent song suggestions
- Find Similar Songs - Discover tracks similar to your favorite songs using Last.fm
- Beautiful UI - Modern glassmorphism design with turquoise and black theme
- Spotify Integration - Direct search links to find songs on Spotify
- Responsive Design - Works seamlessly on desktop and mobile devices

## Tech Stack

- Backend: Flask (Python)
- AI/LLM: OpenRouter API
- Music Data: Last.fm API
- Frontend: HTML, CSS, JavaScript
- Styling: Glassmorphism design pattern
